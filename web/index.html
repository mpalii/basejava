Финальный проект стажировки JavaOPS по курсу Basejava

Краткое описание курса Basejava:

  • средства разработки и технологии: Java 8, IntelliJ IDEA, GitHib/Git, Сервлеты, JSP, JSTL, Tomcat, JUnit, PostgreSQL, GSON, JAXB
  • хранение резюме:
    • в памяти на основе массива, отсортированного массива, списка и ассоциированного массива (Map)
    • в файловой системе (File API и Java 7 NIO File API)
      • в стандартной и кастомной сериализации Java
      • в формате JSON (Google Gson)
      • в формате XML (JAXB)
    • в реляционной базе PostgreSQL
  • деплой веб приложения
    • в контейнер сервлетов Tomcat
    • в облачный сервис Heroku
